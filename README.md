## Project Overview
This project analyzes customer churn patterns for a telecom company
to identify key drivers of customer attrition and support retention strategies.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Calculated Fields
- Data Visualization

## Key Metrics
- Overall churn rate: **26.86%**
- Competitor-related churn: **37.64%**
- Senior citizen churn rate: **40%**

## Analysis Performed
- Customer segmentation by age groups
- Churn analysis by contract type and usage patterns
- Identification of primary churn reasons

## Outcome
Developed an Excel dashboard summarizing churn rate, customer demographics,
and key churn drivers to support data-driven decision-making.
